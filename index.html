Privacy Policy

Effective date: December 11, 2025

Overview. The Kazakhstan Local Sites Enhancer (the “Extension”) is a browser extension that provides a dark theme and page layout improvements for supported Kazakh websites. The Extension is designed to run entirely in the user's browser.

Data collection

The Extension does not collect, transmit, store, or process any personal data. No analytics, tracking, or remote logging services are used.

Local processing

All functionality — including theme changes and removal of non-essential page elements — is performed locally in your browser. The Extension does not send page content or any other information to external servers.

Permissions

The Extension may request standard browser permissions necessary to modify the appearance of supported websites (for example, access to run content scripts on matched pages). These permissions are used solely to apply styling and layout changes and are not used to collect data.

Third-party services

The Extension does not use third-party services, SDKs, or external APIs that would cause data to leave the browser.

Changes to this policy

If this policy changes, a revised version with an updated effective date will be posted at this URL.
